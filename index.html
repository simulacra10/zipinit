ZIP Apportionment Initiative

This site exists to present a single institutional argument:

Legislative representation should be apportioned by ZIP code rather than population.

The ZIP Apportionment Initiative is a volunteer effort devoted to articulating, defending, and advancing this proposal, beginning with the State of Maryland.

This is not a permanent organization. It exists only to pursue clearly defined institutional outcomes and will dissolve once those outcomes are achieved or definitively foreclosed.
